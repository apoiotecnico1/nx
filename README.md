# Pontos Próximos (Auto-GPS + WhatsApp)

- Página em HTTPS (GitHub Pages) que inicia o **GPS** automaticamente e exibe os **10 IDs** mais próximos.
- Importa **CSV, XLS/XLSX, KML, KMZ** (carregados pelo usuário no aparelho).
- Botão **“Enviar IDs no WhatsApp”** monta mensagem com posição + top 10.

## Como usar
1. Abra a página publicada pelo GitHub Pages e **permita a localização**.
2. Clique em **📥 Importar** e selecione o arquivo com os pontos.
3. Veja os **10 mais próximos** e use **Enviar IDs no WhatsApp**.

> CSV recomendado: separador `;` e decimais com `,`. Linhas com `0,0` são ignoradas.
